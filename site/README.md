# Lattice — Agent Headquarters (Website)

This is the marketing site for [latticeruntime.com](https://latticeruntime.com), built with [Next.js](https://nextjs.org).

Part of the [Lattice ecosystem](../../README.md) — the open-source runtime for AI agents.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the site.

## Deployment

Deployed on Vercel. See [Next.js deployment docs](https://nextjs.org/docs/app/building-your-application/deploying) for details.
